Criar um repositorio no Github, você também deve criar uma conta nele  
Lembrando que ao criar o repositorio o github já fornece uma documentação referente a como dar commit  
Indi para o git, vou me referir apenas ao linux  
Você deve baixar o git, o que facilmente pode ser feito digitando sudo apt install git  
configurar o git digitando o git config --global user.name "Seu Nome" -- para configurar seu nome e git config --global user.email "seu@email.com" -- para configurar seu email  
git init para iniciar o git  
git add .  
git commit -m "segundo commit"  
git remote add origin git@github.com:HenriqueeK/Desenvolvimento_web.git ou o repositorio em questão  
git push -u origin main
