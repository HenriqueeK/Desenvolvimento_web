Apenas uma amostra

  • Criar um repositorino no GitHub
  • Criar uma pasta para o repositorino local
  • Istalar git
  • Pelo terminar ir até a pasta e dar git init
  • Criar arquivos, modificar, estilizar etc
  • git add . para adicionar todos os arquivos da pasta, ou git add nome do arquivo
  • git commit -m "mensagem"
  • git remote add origin git@github.com:HenriqueeK/Desenvolvimento_web.git para subir ao github
