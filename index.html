Apenas uma amostra

  • Criar um repositorio no GitHub no site: https://github.com/
  • Lembrando que é necessario criar uma conta no GitHub
  • Criar uma pasta para o repositorio local -- Lembrando que ao criar o repositorio já oferece uma documentação para fazer o commit
  • Também é necessario instalar o git, caso não tenha
  • No linux é só digitar sudo apt install git pelo terminal
  • git config --global user.name "Seu Nome" -- para configurar seu nome
  • git config --global user.email "seu@email.com" -- para configurar seu email
  • Pelo terminal ir até a pasta e dar git init
  • Criar arquivos, modificar, estilizar etc
  • Digitar o git add . para selecionar e incluir todos os arquivos da pasta, ou git add nome do arquivo
  • git commit -m "mensagem"
  • git remote add origin git@github.com:HenriqueeK/Desenvolvimento_web.git para vincular ao github, não sendo necessario indicar toda a vez que faz commit
  • git push -u origin main -- para subir para o github
